# TripCanvas

TripCanvas turns saved travel inspiration into a visual trip planning experience.

## Language

**Trip**:
A visual travel planning experience for one destination, combining saved inspiration, extracted places, itinerary days, and agent recommendations.
_Avoid_: Journey, board, canvas

**Planner Output**:
The backend-generated itinerary narrative and recommendations derived from extracted places and user preferences.
_Avoid_: Trip, frontend state

**Extracted Place**:
A real-world place identified from saved travel inspiration, with enough location information to appear on the map.
_Avoid_: Pin, marker, venue
